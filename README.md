## 🛸 About The Project | О Проекте

**Cardora AI** — это высокотехнологичный комплекс для автоматизации E-commerce процессов. Система объединяет компьютерное зрение (CV), нейросети (Super-Resolution) и табличную автоматизацию для полного цикла создания контента.

> ⚠️ **Note:** Исходный код является коммерческой тайной (Closed Source). Данный репозиторий служит витриной продукта и центром дистрибуции обновлений.

### ⚡ Key Features (Возможности)

| Feature | Description |
| :--- | :--- |
| **🧠 Neural Upscaling** | Интеграция **Real-ESRGAN (Vulkan)**. Увеличение изображений в 400% без потери качества. Умный тайлинг для защиты GPU. |
| **🎨 Photoshop Engine** | Прямое управление Adobe Photoshop через COM-интерфейс. Генерация сложных PSD-макетов и ресайз под маркетплейсы за секунды. |
| **📊 Excel Core** | Двусторонняя синхронизация с Excel. Автоматическое заполнение спецификаций WB/Ozon на основе данных из изображений. |
| **🛡️ Security System** | Привязка лицензии к **HWID** (железу ПК). Защита от декомпиляции (Nuitka Compilation). |
| **🖥️ Fluent GUI** | Современный интерфейс на **PySide6** (Qt) в стиле Windows 11 с поддержкой темной темы и анимаций. |

  <img src="https://skillicons.dev/icons?i=python,qt,pytorch,opencv,ps&theme=dark" />

*   **GUI:** PySide6 (Qt for Python)
*   **AI Backend:** Real-ESRGAN (ncnn implementation)
*   **Automation:** Win32Com (Excel/Photoshop API)
*   **Build System:** Nuitka (Compiled to C++ code)

## 📥 Installation (Установка)

Приложение поставляется в виде скомпилированного инсталлятора.

1. Перейдите в раздел [**Releases**](../../releases).
2. Скачайте последнюю версию `CardoraSetup_v2.5.exe`.
3. Запустите установщик.
4. После запуска скопируйте ваш **HWID** и отправьте администратору для получения лицензионного ключа.
